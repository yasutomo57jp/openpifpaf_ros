# OpenPifpaf ROS node

## installation

```shell
pip install openpifpaf
```

## usage

```shell
ROS_NAMESPACE=camera rosrun openpifpaf_ros openpifpaf_ros_node.py
```

## parameters
|--------- |------------|
|topic name| default    |
|--------- |------------|
|in_topic  | image_color|
|out_topic | human_pose |
|scale     |         1.0|
|--------- |------------|
